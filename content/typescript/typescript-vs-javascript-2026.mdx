JavaScript is the language browsers execute. TypeScript adds a type system that checks your code before it reaches users.

## Learn the runtime first

Understand values, functions, objects, async work, and the DOM. Those skills transfer directly to TypeScript.

## Add types as your projects grow

Types make refactoring safer and give editors better autocomplete. They are especially valuable in shared codebases.
