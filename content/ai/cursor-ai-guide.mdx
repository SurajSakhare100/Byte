Cursor can help inspect a codebase, propose edits, and explain unfamiliar code. Its value grows with precise context.

## Give a bounded task

Name the files, constraints, and desired outcome. Ask for a plan before asking for a large edit.

## Review every change

Treat generated changes as a draft. Check the diff, run tests, and confirm behavior in the application.
