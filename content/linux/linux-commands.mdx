The shell is a fast interface for inspecting and changing a system. Learn a few commands deeply before memorizing a long list.

## Navigate and inspect

Use `pwd`, `ls`, `cd`, `find`, and `rg` to understand where files live and what they contain.

## Respect destructive commands

Read the command before pressing enter. Prefer previews and version control when changing or deleting files.
