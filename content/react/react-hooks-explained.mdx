Hooks let function components remember values and synchronize with the browser. Start with `useState` for information that changes on screen.

## Use state for changing UI

Keep state small and specific. A boolean for an open menu is clearer than one large object for every UI concern.

## Use effects for external systems

Use `useEffect` to connect to subscriptions, timers, or browser APIs. Values calculated from props and state usually do not need an effect.
