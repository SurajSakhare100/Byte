Array methods help you express collection work without manual loops. The most useful distinction is whether a method creates a new array or changes the original one.

## Transform with map

`map` returns one new value for every input item. Use it to produce display data or reshape API results.

## Select with filter

`filter` keeps only items that pass a condition. It is ideal for search, permissions, and status views.
