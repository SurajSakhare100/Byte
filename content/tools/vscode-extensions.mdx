Extensions should reduce friction, not turn your editor into a dashboard. Start with a formatter, linter, Git integration, and language support.

## Essential categories

Use Prettier for formatting, ESLint for feedback, GitLens for history, and error highlighting for fast debugging.

## Keep the list intentional

Review extensions every few months. Remove tools you no longer use to keep the editor quick and predictable.
